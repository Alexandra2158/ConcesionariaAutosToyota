🔄 Flujo Principal de Negocio

Cliente HTTP→ POST /api/Auth/login→ LoginUserHandler (COMMAND)→ UserRepository → SQL Server→ JWT Token (roles + claims)
Bearer Token→ POST /api/Order→ CrearOrderHandler (COMMAND)→ OrderRepository → SQL Server→ 📤 OrderCreatedEvent → RabbitMQ
📥 OrderCreatedConsumer→ Descuenta Stock en SQL Server→ Log de auditoría
